# PerformanceTestAutomations
PerformanceTestAutomations provide Automation solutions of Performance testing
It reads HTML that generated from LoadTest and add customised SLA & SLAMET coulmns in PAGES table and later renamed to Transactions Summary Table
And it adds Over all SUmmary table you can add the data which you want to present in that table.
In case of test fail it will add  Transactions that need Attention table with Transactions name,Avg response time, SLA of that transaction and Error rate of that transaction.
after customising HTML report it will customise the details to Email body with Over ALL Summary Table & Transactions that need attention into email body & send to email for repsective distro or individuals that you want to share.
